# Tugobo AI

AI reservation and lead conversion system for hotels, via WhatsApp.

## Quick Start

### 1. Install dependencies

```bash
pnpm install
```

### 2. Set up environment

```bash
cp .env.example apps/web/.env.local
```

Then fill in `apps/web/.env.local` with your real keys (see step-by-step below).

### 3. Set up Supabase

1. Create a free project at [supabase.com](https://supabase.com)
2. Copy your **Project URL** and **anon key** from Project Settings → API
3. Copy the **service_role key** (keep this secret)
4. Copy the **direct connection string** from Project Settings → Database → Connection string (NOT the pooler)
5. Add all four values to `apps/web/.env.local`

### 4. Set up Twilio WhatsApp Sandbox (Day 1)

1. Create a free account at [twilio.com](https://twilio.com)
2. Go to Messaging → Try it out → Send a WhatsApp message
3. Note your **Account SID** and **Auth Token** from the Console Dashboard
4. The sandbox number is `+14155238886`
5. Add to `.env.local`:
   ```
   TWILIO_ACCOUNT_SID=ACxxxxxxxx
   TWILIO_AUTH_TOKEN=your-token
   TWILIO_WHATSAPP_NUMBER=+14155238886
   ```

### 5. Get your webhook URL (ngrok)

Twilio needs a public URL to send webhooks to your local machine.

```bash
# Install ngrok: https://ngrok.com/download
ngrok http 3000
# Copy the https URL, e.g. https://abc123.ngrok-free.app
```

Set in `.env.local`:
```
NEXT_PUBLIC_APP_URL=https://abc123.ngrok-free.app
```

Then in Twilio Sandbox settings, set the webhook URL to:
```
https://abc123.ngrok-free.app/api/webhooks/twilio
```

### 6. Add your OpenAI key

Get from [platform.openai.com/api-keys](https://platform.openai.com/api-keys):
```
OPENAI_API_KEY=sk-...
```

### 7. Run the dev server

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

### 8. Test the end-to-end pipe

1. Join the Twilio sandbox (send the join code from the Twilio console to `+14155238886` on WhatsApp)
2. Send any message (e.g. "Hi, do you have rooms for July?")
3. You should get an AI reply within 5 seconds

## Database migrations

After filling in `DATABASE_URL`:

```bash
pnpm db:push    # push schema directly (dev)
# or
pnpm db:migrate # run migration files
```

## Project structure

```
tugobo-ai/
├── apps/web/              # Next.js 15 dashboard + API
├── packages/
│   ├── shared/            # Types, logger, env
│   ├── db/                # Drizzle schema + client
│   ├── channels/          # WhatsApp adapters (Twilio / Meta)
│   └── core/              # Agents, tools, Inngest workflows, prompts
├── supabase/migrations/   # Generated by Drizzle
├── .cursor/               # Cursor AI rules + architecture docs
├── .env.example           # Copy to apps/web/.env.local
└── AGENTS.md              # AI agent guidelines
```
